酷文奇题应用 使用指南

发表于 2021-07-20 | 分类于 酷文奇题应用专题 | 阅读次数:

酷文奇题是一款专注于iOS开发学习交流的应用，其功能主要可以分为5大块，分别为：

  • 酷奇专题
  • 酷文共赏
  • 奇题共解
  • 番茄时钟
  • 我的汗水

应用以Tab的形式对着5大块内容进行分别的呈现，您可以点击Tab来切换至对应的功能。如图：

酷奇专题

开发者根据自己的经验，按iOS开发所需要的知识体系，整理了一系列博文和面试题。您可以通过点击右上角的列表图标，或者在手机屏幕左侧进行侧滑，拉出知识体系列表，选择您感兴趣的体系内容进行阅读。如图：

知识体系的内容大致分为：

  • 开发语言 Objective-C
    • 内存管理
    • Runtime
    • Blocks
    • 并发编程
    • Runloop
  • COCOA TOUCH FOUNDATION
    • Fundamentals
    • File and Data Persistence
    • Networking
  • COCOA TOUCH UIKIT
    • App Structure
    • App User Interface
    • App User Interactions
  • COCOA TOUCH APP SERVICE
    • Core Animation
    • WebKit
  • COCOA TOUCH MEDIA
    • AVFoundation
    • HTTP Live Streaming
  • 架构
    • 第三方框架
    • UML和设计模式
    • 架构
  • 性能与安全
    • 性能优化
    • 安全
  • 调试，测试与持续集成，开发工具
    • 调试
    • 测试与持续集成
    • 开发工具
  • 计算机软件基础
    • 数据结构与算法
    • 编程思想
    • 数据库

每个知识体系列表的内容又为大家编写了导读，并按照核心阅读，扩展阅读，你问我答的形式，为大家推荐和总结了了一些博客文章和面试问题。如图

您可以通过点击来阅读这些博文或者面试问题的详细内容。

酷文共赏

酷文共赏Tab为您展示了用户推荐的酷文，每一个Cell为您展示了推荐博文的标题，简介或者推荐理由，推荐人，该推荐内容的热度（评论数，喜欢数，分享数）以及所属的标签。

您可以进行如下的交互：

  • 点击左上角的按钮，按您喜欢的方式对博文进行排序
  • 点击标题，查阅博文的具体内容
  • 点击推荐人，查阅他推荐的其他内容
  • 点击评论，查阅其他用户对推荐内容的评论
  • 点击标签，您可以查阅更多和该标签相关的推荐的博文
  • 点击分享，您可以将该博文进行分享

此外，如果您登录后，还可以

  • 点击评论，添加您对这篇推荐博文的评论
  • 点击喜欢，该博文添加到您的喜欢列表
  • 推荐博文

奇题共解

奇题共解Tab为您展示了用户提出的问题列表，每一个Cell为您展示了推荐奇题的内容推荐人，该奇题的热度（评论数，喜欢数，分享数）以及所属的标签。

您可以进行如下的交互：

  • 点击左上角的按钮，按您喜欢的方式对奇题进行排序
  • 点击回答，查看其他用户对这个问题的回答
  • 点击推荐人，查阅他推荐的其他内容
  • 点击标签，您可以查阅更多和该标签相关的奇题
  • 点击分享，您可以将该奇题进行分享

此外，如果您登录后，还可以

  • 点击回答，添加您对这个奇题的回答
  • 点击收藏，将该奇题添加到您的收藏列表
  • 添加奇题

我的汗水

我的汗水Tab为您提供了个人信息的管理，您可以通过点击“酷奇”的头像进行登录/注册/找回密码。

登录后，该页面会展示您的个性化页面，包括您推荐的酷文，发表的奇题，以及您的评论，回答，喜欢，收藏，以及消息

番茄时钟

番茄时钟Tab为您提供了番茄工作法的工具，来帮助您更好的专注于学习。您可以设定您需要专注的内容，专注的时间，并开始专注。番茄时钟会为你开启一段时间的专注，并为您统计您的专注时间。

酷文奇题APP使用指南就简单介绍到这里，欢迎前往Apple Store查询“酷文奇题”进行下载，并希望您确实能从这个APP中收获满满，提升自己的开发技能。
