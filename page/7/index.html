Key-Value Coding Programming Guide

发表于 2018-05-09 | 分类于 斑马日报 2018/05/09 第9期 | | 阅读次数:

适合: 初级开发者阅读
来自: developer.apple.com
关键字: Key-Value Coding, KVC
原文：Key-Value Coding Programming Guide

Key-Value Coding, KVC, 键值编码，这是我们在日常编码中经常会用到的一种技术。这得益于OC的语言特性，可以让我们不必进行任何操作就可以进行属性的动态读写。KVC的操作方法由NSKeyValueCoding非正式协议提供，而NSObject以Category的形式实现了这个非正式协议，因此可以说OC中几乎所有的对象都支持KVC操作，正如你常用到的， KVC常用的操作方法如下：

  • 动态设置：setValue:forKey:, setValue:forKeyPath:
  • 动态读取：valueForKey:, valueForKeyPath:

那么你知道它的实现原理吗？它还有哪些别的功能吗？又或者我们在使用KVC编码时需要注意什么？它的性能又如何？Apple给出了答案，那就是推荐给大家的Key-Value Coding Programming Guide 。

不知道为什么Apple开始废弃一系列Programing Guide的更新。在新版的developer.apple.com，我们已经找不到这篇文章了，我们可以在archive中找到，但这并不是说这篇文章过时了，他依旧很有指导意义，帮助你了解KVC背后的故事。所以，如果你的英文还可以的话，那么这是你必须要阅读的关于KVC的第一篇文章。
